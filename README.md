## SETUP
```
npm install
```

## 1/ START
### 1.1/ chạy thường
```
npm run start
```
### 1.2/ chạy scraping
```
npm run start_scraping
``` 

## 2/ PM2
### 2.1/ chạy pm2 với 4 cluster
```
npm run pm2
```
### 2.2/ xóa pm2
```
npm run pm2_delete
``` 

## 3/ DEV
### 3.1/ chạy môi trường dev
```
npm run dev
```
### 3.1/ chạy scraping môi trường dev
```
npm run dev_scraping
```

## 4/ FLUSTER
fluster tự setup
```
npm run dev_cluster
```
